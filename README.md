# raccoon-msg
Random string messages from a raccoon

Whenever I start a new project, I don't know what to write for the first commit. After doing a “git init” there is technically nothing there...

## Install

```npm
npm install -g raccoon-msg
```

# Usage

```bash
 raccoon-msg
```
